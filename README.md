# ORIGYN Website

Project repository for the ORIGYN website.

## Current status

Phase 2 (Design System Creation) implementation now includes:
- Tokenized brand system (color, typography, spacing, motion)
- Responsive 12/8/4 grid primitives
- Reusable component library for layout, CTA controls, cards, architecture blocks, industry tiles, code snippets, and technical panels
- Interactive navbar (scroll + mobile menu states)
- Functional code snippet copy interaction and lightweight syntax styling

See `DESIGN_SYSTEM.md` for component APIs and governance rules.
